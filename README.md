# README
As of April 2026, I have officially pulled and archived all of my open-source code from GitHub.

If you are looking for my past projects—whether it's ESP32, MeshCore stuff, or amateur radio — they are no longer publicly available here.

Why?

Because technologically illiterate politicians have decided to legislate things they fundamentally do not understand. Specifically, California’s AB 1043 (Digital Age Assurance Act) and Colorado’s SB 26-051 (Age Attestation on Computing Devices) have made it a massive legal and financial liability to simply share code.  

These profoundly ignorant laws mandate that software and operating system developers integrate age-verification mechanisms into their systems to broadcast an "age-bracket signal" via API to any app that requests it. They are threatening developers with fines of up to $7,500 per affected user for non-compliance.  

These mandates were clearly drafted by out-of-touch lawmakers who still need their grandkids to help them connect to Wi-Fi, yet they feel qualified to dictate the architecture of global open-source software. They have no concept of how decentralized, volunteer-run projects like Linux distributions, mesh networks, or microcontroller firmware actually work.

I refuse to comply with this knee-jerk, moronic legislation. Mandating the collection of personal user data under the guise of "safety" is an absolute joke, and it forces developers to implement what is, by definition, the first step toward state-mandated spyware.

I build tools to solve problems and share knowledge freely.

I do not write code to act as an unconsenting data broker or a surveillance proxy for lawmakers who are utterly clueless about technology.

The open-source community thrives on privacy, freedom, and the open exchange of ideas.

Forcing independent developers to build surveillance APIs to harvest identifying information from their users is a complete violation of those principles.

Until the clowns writing these laws acquire a foundational understanding of the internet and stop treating independent open-source developers like multi-billion-dollar data-harvesting corporations, my code will remain private.
